# Description :

    This is a Api built in express.js for the twitter clone app built in  React.js

### cloning :

    git clone https://github.com/firminfinva/twitter-clone-express.git

### Install dependencies:

    cd twitter-clone
    npm install

# Run the application:

    npm run dev
